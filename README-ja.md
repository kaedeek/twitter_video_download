# Twitter_Video_Download

X（旧Twitter）の動画をダウンロードするためのスクリプトです。

English [here](https://github.com/kaedeek/twitter_video_download/blob/main/README.md)

## 使い方

- **1.** **インストール手順**
```
git clone https://github.com/kaedeek/twitter_video_download.git
cd twitter_video_download
```

- **2.** **実行方法**
```
cargo run -- https://x.com/~
```
